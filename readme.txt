= What is Open Graph? =

Open Graph is used by some WebPages to agregate Page-Informations. For Excampe <a href="http://developers.facebook.com/docs/opengraph/"Facebook</a>.

= What is a QR-Code? =

A QR-Code is a machine readable Image that encode Informations like a Weburl.
User with readable devices like a smart phone can open the url direct in there device.
